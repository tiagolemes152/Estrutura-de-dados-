# Sistema de Fila de Atendimento

Este projeto é um sistema de fila de atendimento desenvolvido em JavaScript.

A aplicação permite adicionar pessoas à fila, visualizar a ordem de atendimento,
buscar um usuário pelo nome e identificar sua posição na fila.

O objetivo é simular o funcionamento de filas reais, como em bancos ou hospitais,
facilitando o controle e organização do atendimento.

## Funcionalidades

- Adicionar pessoa à fila
- Remover pessoa da fila
- Listar todos da fila
- Buscar posição na fila

  ## Tecnologias

- HTML
- CSS
- JavaScript

  ## Como executar

1. Clone o repositório
2. Abra o arquivo index.html no navegador
